# Leaked passwords / credentials

**Priority: HIGH — credential reuse is the most common path to account takeover.**

## Do this now
- Change the password on the breached site, and **everywhere you reused it**. Use a long unique passphrase (16+ characters). [1]
- Turn on multi-factor authentication (MFA); prefer an authenticator app or hardware key over SMS. [2]

## Do this soon
- Start using a password manager so every account has a unique password. [1]
- Check which breaches your email appears in. [3]

## Keep doing
- Periodically re-check exposure and rotate any password that shows up again. [3]
