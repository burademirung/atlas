# Leaked driver's license

**Priority: MEDIUM — a license is widely used to verify identity.**

## Do this now
- Contact your state DMV; ask whether they can flag the record or issue a new license number. [1]

## Do this soon
- Place a fraud alert or credit freeze, since the license can be used for identity verification. [2]

## Keep doing
- Watch for accounts or citations opened in your name.

## Sources
- [1] FTC IdentityTheft.gov — https://www.identitytheft.gov/
- [2] CFPB — credit freeze / fraud alert — https://www.consumerfinance.gov/ask-cfpb/what-is-a-credit-freeze-or-security-freeze-on-my-credit-report-en-1341/
