# Business: leaked customer / employee data or secrets

**Priority: CRITICAL — legal notification clocks may already be running.**
*General information, not legal advice — engage counsel and your cyber-insurance carrier early.*

## Do this now
- Preserve evidence; do not wipe affected systems. Engage breach counsel and your cyber-insurance carrier — they coordinate the response. [1]
- If source code or secrets leaked: rotate all exposed API keys, tokens, and credentials; invalidate sessions; scan history for more secrets. Assume anything exposed is compromised.

## Do this soon
- Determine notification obligations — the **shortest applicable deadline governs**. See `notification_laws.md`. [2][3]
- Prepare individual, regulator, and (where required) media notifications.

## Keep doing
- Document everything for claims and regulators; run a post-incident review and harden the gaps.
